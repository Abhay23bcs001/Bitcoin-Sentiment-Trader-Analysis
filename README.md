# Bitcoin Market Sentiment vs Trader Performance Analysis

## Project Overview

This project investigates the relationship between Bitcoin market sentiment and trader performance using:

* Bitcoin Fear & Greed Index
* Hyperliquid Historical Trader Data

The objective is to identify patterns between market sentiment and trading outcomes such as profitability, win rates, trading volume, and trader behavior.

## Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

## Methodology

1. Data Cleaning
2. Data Transformation
3. Date Standardization
4. Dataset Merging
5. Exploratory Data Analysis
6. Sentiment-Based Performance Analysis
7. Visualization
8. Insight Generation

## Key Findings

* Extreme Greed produced the highest average profit per trade.
* Fear generated the highest cumulative profitability.
* Extreme Greed achieved the highest win rate.
* Extreme Fear showed the weakest trading performance.
* Fear generated the highest trading volume.
* SELL trades dominated during Greed periods.
* Extreme Greed exhibited the highest volatility.
